## get started

```shell
npm install chimee-helper-log --save
```

if you are using `flow`, you should import our flow defination, by adding this to your `.flowconfig`.

```
[ignore]

[include]

[libs]
./node_modules/chimee-helper-log/lib/index.flow.js
[options]

[lints]
```
